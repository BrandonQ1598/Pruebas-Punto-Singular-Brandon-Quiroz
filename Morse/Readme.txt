Este archivo puede ser ejecutado desde la línea de comandos de windows siguiendo las siguientes instrucciones: 
1.- Ejecutar el cmd 
2.- Ingresar a la ruta donde esta guardado en archivo, ejemplo: " cd C:\Users\branq\Desktop>" 
3.- Identificar la ruta en la que se encuentra el compilador de java "javac.exe" p.ej. "C:\Program Files\Java\jdk1.8.0_161\bin\javac.exe" 
4.- Copiar la ruta anterior en cmd seguido del nombre del archivo y su extensión p.ej. "C:\Program Files\Java\jdk1.8.0_161\bin\javac.exe" morse.java 
5.- Escribir 'java' seguido del nombre del archivo, respetando mayusculas y omitiendo la extensión p.ej. java Morse 
6.- El programa empezará a ejecutarse, pedirá que se ingrese una palabra o frase, que podrá contener letras o números o letras en codigo morse 
7.- Una vez escrita la palabra o frase se presiona el boton ENTER y el programa se encargará de traducir el mensaje escrito.
